# REmember

A compact, side-by-side companion app built to store code snippets and quick notes while you work in your IDE. Live at **[remember.ves3.no](https://remember.ves3.no/)**.

---

## Features

* **Code vs. Note Toggle:** Switch seamlessly between formatted text notes and code mode.
* **Syntax Highlighting:** Full syntax highlighting for clean, readable code snippets.
* **Instant Search:** Quickly filter through saved notes and snippets.
* **Ultra-Compact UI:** Optimized specifically for phones and narrow PC windows—ideal for keeping snapped to the side of VS Code without taking up valuable screen space.

---

## Getting Started

### Access the Web App
Visit the live site directly:  
👉 **[https://remember.ves3.no/](https://remember.ves3.no/)**

### Local Setup & Development

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Ves3-no/Remember.git](https://github.com/Ves3-no/Remember.git)
   cd Remember

2. **Install dependencies:**
npm install

3. **Run the development server:**
npm run dev

## Recommended Layout Setup
Remember is designed to run in a narrow viewport so you don't lose screen real estate while coding:

| Main Workspace (~70%) | Side Companion (~30%) |
| :--- | :--- |
| **VS Code** | **Remember** |
| Active code editing | Quick code snippets & notes |
